              Description

              I suggest it looks like this (logo and text centered):

              <pico container logo as on homepage>
              I was expecting a paradigm shift
              and all I got was a lousy constructor

              Before the T-Shirts can be made, we need:

              1) An SVG of the PicoContainer logo so we can scale it for the T-Shirts.
              2) Get the font that matches the one in the logo. (Will be used for text).

              If you want one, just add a comment with:

              • name
              • email
              • preferred colour (black/white)
              • size (S,M,L,XL,XXL)

              We'll get back to everyone later about pricing and delivery.
